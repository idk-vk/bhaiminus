### Install using
npm i -g bhaiminus

to check if it's working write on cmd
bhaiminus -v

then open a folder,
create a .bpp file

then install bhaiminus locally using

npm i bhaiminus

now to execute the file run
bhaiminus <filename>

for eg
bhaiminus anshu.bpp


Here is an example bpp code
 
bolo bhai('Hi bhai.');

notice the error after running this code.

now run proper code 
bol bhai('hi bhai');
